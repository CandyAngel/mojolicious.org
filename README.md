## Working Locally

  Clone the repository.

    $ git clone git://github.com/kraih/mojolicio.us.git
    $ cd mojolicio.us

  Install [Mojolicious](http://mojolicio.us).

    $ curl get.mojolicio.us | sh

  Start hacking.

    $ morbo mojolicious.pl
