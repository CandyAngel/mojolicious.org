The L<Mojolicious> website.

=head2 Working Locally

Clone the repository.

    git clone git://github.com/kraih/mojolicio.us.git
    cd mojolicio.us

Install L<Mojolicious>.

    curl -L cpanmin.us | perl - http://latest.mojolicio.us

Start hacking.

    ./mojolicious.pl daemon
